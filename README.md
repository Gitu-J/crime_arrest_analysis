## Project Overview
This project investigates crime patterns and predicts arrest outcomes using big data analytics and machine learning techniques.

Data preprocessing, feature engineering, and model development were conducted in Google Colab using Python. The processed dataset was then visualized in Tableau to generate interactive dashboards for analytical interpretation.

## Machine Learning Models Implemented
The following models were developed and evaluated:

- Logistic Regression  
- Decision Tree Classifier  
- Random Forest  
- Gradient Boosted Trees  
- K-Means Clustering (Unsupervised Learning)

These models were used to analyse crime characteristics and predict the likelihood of arrest outcomes.
