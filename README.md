## Project Overview
This project analyzes crime patterns using Machine learning and big data.
The dataset was prepared in Excel and visualized using interactive dashboards.

## Files Included
- tableau/crime_dashboard.twbx – Tableau packaged workbook
- tableau/crime_sample.xlsx – Cleaned dataset used for analysis

## Dashboard Insights
1. Hourly crime trends show peak activity during afternoon and evening hours.
2. Crime type distribution varies across clusters.
3. Arrest prediction shows differences between cluster groups.
